HTML Canvas Tutorial

What is the HTML Canvas Element?

The <canvas> element is used to draw graphics, animations, and other visual elements using JavaScript. It allows developers to dynamically generate images, shapes, text, and animations directly in the browser.

How to Use the Canvas Element?

To use the canvas element, follow these steps:

  1. Create a <canvas> element in HTML.
  2. Access the canvas with JavaScript using `getContext("2d")`.
  3. Use various canvas methods to draw shapes, text, or apply styles.

Basic Canvas Syntax

<canvas id="myCanvas" width="400" height="300"></canvas>

<script>
    let canvas = document.getElementById("myCanvas");
    let ctx = canvas.getContext("2d");
    ctx.fillStyle = "blue";
    ctx.fillRect(50, 50, 200, 150);
</script>

Circle

<script>
    let canvas = document.getElementById("circleCanvas");
    let ctx = canvas.getContext("2d");
    ctx.beginPath();
    ctx.arc(150, 150, 80, 0, Math.PI * 2);
    ctx.fillStyle = "#3498db";
    ctx.fill();
    ctx.stroke();
</script>

Rectangle

<script>
    let canvas = document.getElementById("rectangleCanvas");
    let ctx = canvas.getContext("2d");
    ctx.fillStyle = "#e74c3c";
    ctx.fillRect(50, 50, 200, 150);
    ctx.strokeRect(50, 50, 200, 150);
</script>

Triangle

<script>
    let canvas = document.getElementById("triangleCanvas");
    let ctx = canvas.getContext("2d");
    ctx.beginPath();
    ctx.moveTo(150, 50);
    ctx.lineTo(50, 250);
    ctx.lineTo(250, 250);
    ctx.closePath();
    ctx.fillStyle = "#2ecc71";
    ctx.fill();
    ctx.stroke();
</script>

Linear Gradient

<script>
    let canvas = document.getElementById("linearGradientCanvas");
    let ctx = canvas.getContext("2d");
    let gradient = ctx.createLinearGradient(0, 0, 300, 0);
    gradient.addColorStop(0, "red");
    gradient.addColorStop(1, "yellow");
    ctx.fillStyle = gradient;
    ctx.fillRect(0, 0, 300, 300);
</script>

Radial Gradient

<script>
    let canvas = document.getElementById("radialGradientCanvas");
    let ctx = canvas.getContext("2d");
    let gradient = ctx.createRadialGradient(150, 150, 10, 150, 150, 100);
    gradient.addColorStop(0, "blue");
    gradient.addColorStop(1, "white");
    ctx.fillStyle = gradient;
    ctx.fillRect(0, 0, 300, 300);
</script>
